# octopus-clicker-game
A fun, interactive pixel art widget built with HTML5 Canvas, CSS, and vanilla JavaScript. 

Click the mystery box to make a cute octopus pop out!

## Features
- **Progressive Mood States:** The octopus transitions from super happy, to annoyed, to completely angry based on your click volume.
- **Lockout Mechanism:** Reaching the click limit slams the box shut for a 1-minute timeout.
- **Reactive Grumbling:** If you attempt to click or poke the box during its 1-minute cooldown, it will violently shake at you for 3 seconds to tell you to back off.
- **Clean Aesthetic:** Beautifully centered layout with zero cluttered text UI elements for a pure widget experience.

## How to Play / Run Locally

Since this is a lightweight frontend website, you don't need to install anything heavy to try it out!

1. **Download or Clone** this repository to your computer.
2. Open the project folder in a code editor like **VS Code**.
3. Use an extension like **Live Server** to run it locally
4. Or just open the `index` file and you're good to go!
